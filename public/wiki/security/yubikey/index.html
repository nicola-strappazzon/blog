YubiKey

09/05/2025

En mi caso personal, escogi esta marca y la serie 5 porque soporta un cifrado RSA hasta 4096 que es lo mas seguro hasta el momento y tiene soporte para OpenPGP, por último tiene soporte con USB C el cual es bastante practico hoy en día.

Se recomienda tener dos dispositivos o más por seguridad, uno de ellos deberá guardarlo en un lugar seguro en caso de perdida.

Existe una cli open source para administrar el dispositivo llamada YubiKey Manager CLI.

Instalar en Arch linux:

sudo pacman -Sy yubikey-manager

Instalar en MAC OS:

brew install ykman
